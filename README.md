# AI Resume Analyzer

## Overview
AI Resume Analyzer is a full-stack project built using FastAPI and React.

Users can:
- Upload Resume PDF
- Paste Job Description
- Get ATS Score
- Get Matching Skills
- Get Missing Skills
- Get AI Suggestions

## Technologies Used
- FastAPI
- React
- LangChain
- Groq API
- Docker
- Vite

## Run Backend

uvicorn app.main:app --reload

## Run Frontend

cd frontend
npm install
npm run dev
